### Work with docker
- To start working environment, please run `docker-compose up -d`
- To access laravel app, access address `http://localhost:8122`
>docker exec -it ai_hotel-api chmod -R 775 storage

>
> Port: 8433
>
> Username: ai_hotel

> Password: ai_hotel
